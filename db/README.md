# Denwire Database Reference

**Database**: `denw` on IBM Informix 7.23.UC13 SE  
**Total tables**: 271 (application tables + Informix system tables)

---

## Module Prefix Guide

Tables are named with a two-letter prefix indicating their ERP module.

| Prefix | Module | Description |
|--------|--------|-------------|
| `ad` | Address / Admin | Address lookup, labels, numbering, terminal defs |
| `ap` | Accounts Payable | Supplier contacts and descriptions |
| `bk` | Bank | Bank accounts, payments, reconciliation, statements |
| `cm` | Cost / Manufacturing | Products, BOMs, routes, stages, operations |
| `cv` | Currency | Exchange rates and currency master |
| `dl` | Debtors / Sales Ledger | Customers, transactions, credit control, reps, teams |
| `dm` | Document Management | Letter templates |
| `gl` | General Ledger | Chart of accounts, transactions, budgets, divisions |
| `gr` | ? (small module) | Detail, qty, rep, selection tables |
| `lc` | Letters of Credit | LC headers, items, tariffs, agents |
| `nd` | Notes / Documents | Notes master and contacts |
| `np` | Notes / Print? | Categories, headers, notes |
| `pl` | Purchase Ledger | Suppliers, transactions, currency, groups |
| `po` | Purchase Orders | Orders, lines, deliveries, GRN, invoices, returns |
| `sa` | Sales Analysis | Invoice fact table, period stats, rep stats (see [modules/SA-Tables.md](modules/SA-Tables.md)) |
| `sm` | Sales / Margin? | Deal pricing, delivery, margin pricing |
| `so` | Sales Orders | Full SO lifecycle (see [modules/SO-Tables.md](modules/SO-Tables.md)) |
| `st` | Stock | Stock master, locations, movements, serials, batches |
| `ts` | ? | Time/schedule list |
| `tx` | Tax / VAT | Tax transactions, rates, registrations, payments |
| `xx` | Work / Temp | Temporary stock and supplier tables |
| `sys` | Informix System | Informix system catalog tables (read-only metadata) |
